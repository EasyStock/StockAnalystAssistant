# StockAnalystAssistant
--------------------


  股票分析助手(StockAnalystAssistant)是一个用python写的辅助工具，具有获取股票数据，分析和测试分析结果等功能，为日常投资提供辅助。
  
  
  
--
# 主要功能
+ 获取历史信息(不复权，后复权)
+ 获取交易日当日
+ 各种分析工具
+ 测试分析的结果



# How to 
## 如何安装:
+ 运行 sudo python ./src/installer/AutoInstaller.py
